Path Operating System

About


PathOS is a 16-bit operating system written in Assembly and Basic. The OS has an image viewer, text viewer, games, a C interpreter, a BASIC interpreter, a GUI, and many other cool features! PathOS is intended for people who want to learn about computers, especially low-level code and operating systems. It uses the MikeOS API for many of its functions.
